# Quote Generator Web App

This is a simple web app that generates motivational quotes based on a topic entered by the user. The app provides an easy way to get inspired by displaying quotes related to themes like fear, success, or failure.

---

## What it does

* Lets users enter a topic into a form.
* Displays three motivational quotes that match the topic.
* Shows a friendly message if no matching quotes are found.
* Built with clean and minimal UI using ShadCN components.

---

## Technologies

* Next.js
* TypeScript
* Tailwind CSS
* ShadCN UI
* pnpm
* Vercel (for deployment)

---

## How to run locally

1. Clone the repository:

   ```
   git clone https://github.com/yourusername/Nexium_Aimen_QuoteGeneratorWebApp.git
   ```

2. Go to the app folder:

   ```
   cd Nexium_Aimen_QuoteGeneratorWebApp/internship/assignment-1
   ```

3. Install dependencies:

   ```
   pnpm install
   ```

4. Start the development server:

   ```
   pnpm dev
   ```

5. Open your browser at:

   ```
   http://localhost:3000
   ```
